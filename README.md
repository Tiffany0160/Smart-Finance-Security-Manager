# 💰 Smart Finance & Security Manager

## 📌 Project Description

The **Smart Finance & Security Manager** is a Python command-line application designed to help users manage their personal finances while promoting basic cybersecurity awareness. The system allows users to log in securely, track income and expenses, monitor their account balance, set savings goals, and detect unusual spending patterns through a simple fraud detection feature.

This project combines programming fundamentals with real-world financial management concepts, making it an excellent portfolio project for beginners and intermediate Python developers.

---

## ✨ Features

- 🔐 Secure login system
- 💰 View current account balance
- ➕ Add income
- ➖ Record expenses
- 📂 Categorize expenses
- 📋 View expense history
- 🎯 Set and monitor savings goals
- 🚨 Detect suspicious transactions
- 📊 Display savings progress
- 🖥️ Interactive menu-driven interface
